# Time Series Anomaly Visualization Tool 
Design and develop an interactive visualization tool for time series anomaly detection that highlights anomalies identified by various ML-based and statistical detection methods. The tool should allow users to select different historical lookback windows (e.g., 24 hours, 48 hours, and 72 hours) and observe how anomaly detection results change depending on the selected context window. 

Z-score with 1 std, 2std, 3std, anomalies will appear in a dropbox. 

The application will ingest time series data from CSV files containing timestamp and value columns and provide visual comparisons of anomaly detection outputs across multiple approaches. The project will help improve understanding of how different models and lookback periods influence anomaly detection behavior in network observability data.
